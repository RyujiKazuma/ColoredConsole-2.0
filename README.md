# ColoredConsole-2.0

# What does it do ?

You will enter an year.
Program will generate 5 years backwards and 5 yeaars forward.
Then it will write down every single day and set colors.

If is leap-year then is the year itself yellow and february is lignt blue.
Normally it will be
* Spring months will be green
* Summer months will be red
* Fall months will be purple
* Winter months will be blue
